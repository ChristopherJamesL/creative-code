# Document Manager

A full-stack document management application with AI-powered summarization.

## Tech Stack

**Frontend** вЂ” React 19, TypeScript, Vite, Tailwind CSS v4, TanStack Query, React Router v8, Axios

**Backend** вЂ” Node.js, Express 5, TypeScript, Supabase, tsx

---

## Getting Started

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm run dev
```
